# Service Card Images

This directory contains featured images for service cards.

## Image Requirements
- Format: JPEG or PNG
- Dimensions: 800x400px (recommended)
- File size: Keep under 500KB for optimal performance

## Suggested Image Themes
- IT Support: Technology, computers, IT support, help desk
- Business/Requisition: Office, meetings, collaboration, business
- Digital Worker: Digital transformation, automation, data, analytics
